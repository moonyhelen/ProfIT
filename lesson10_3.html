Создайте класс Vector для представления вектора в

трехмерном пространстве (свойства для координат x, y, z).

Добавьте в прототип Vector для метода plus() и scalar() для

вычисления суммы двух векторов и скалярного произведения

двух векторов. Добавьте в прототип свойство только для

чтения length, подсчитывающее длину вектора.

Переопределите в классе vector методы toString() и valueOf().

Протестируйте созданный класс.
